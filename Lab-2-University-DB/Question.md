# Lab 2 - University Database Tasks

## Tasks

1. Construct a new university database by following the given schema diagram.

2. Add 4 valid data and 2 invalid data to each table. Check whether your database design is discarding invalid data properly or not. [Include the error message in your submitted file.]

3. Display all values of the student table.

4. Display all values of the course table.

5. Display all values of the department table.

6. Display all values of the result table.

7. Display only the student ID and department of the students.

8. Display only the course code and department of the courses.

9. Display only the student ID, course code, and grade of the students.

10. Display only the department name, building, and establish year of the departments.
